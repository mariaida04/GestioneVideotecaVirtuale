# GestioneVideotecaVirtuale
Progetto "Gestione di una collezione di film" relativo al corso di Ingegneria del Software, A.A. 2024/2025.

# Descrizione
Implementazione di un'applicazione in Java per la gestione di una collezione personale di film. 
All'utente sono consentite le operazioni di aggiunta, modifica, rimozione, ricerca, filtraggio e ordinamento di film.
